## 数据说明

古诗词数据按朝代存储于多个 csv 文件中，有“题目”、“朝代”、“作者”和“内容”共四个字段。分成多个文件是为了避免单个文件大于 30M。

古诗词中有一些生僻字，这些生僻字属于 utf8mb4 字符，在许多设备中无法显示，故而使用“?”来替代。
